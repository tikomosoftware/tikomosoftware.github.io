# Prism ガイドツアー ── 順に辿れば「4つの声」と300枚が腑に落ちる

300枚のギャラリーは宝の山ですが、いきなり開くと迷子になります。これは**順路**です。上から辿れば、
言語の考え方（事実・変換・作用・失敗＋時間）を体得しながら、「その考え方で何が作れるか」を
最短で味わえます。各停留所に**厳選した数枚**と、それぞれ「**何に注目するか**」を添えました。

## 使い方
1. `prism serve` を実行し、ブラウザで `playground.html` を開く（Pyodide の読み込みを少し待つ）。
2. 各停留所の **`検索: xxx`** を遊び場の検索ボックスに打ち、★の項目を選ぶ。
3. 右のボタン ── **Run**（コンソール出力）／**Draw**（静止画）／**Animate**（アニメ）── を押す。
4. 🌈 **vivid** トグルを入れると、光りもの・エフェクトが映えます。

> 言語そのものを手で学ぶなら、先に [TUTORIAL.md](TUTORIAL.md)（hello → 能力まで6課）と
> [OVERVIEW.md](OVERVIEW.md)（思想と使い方）をどうぞ。このツアーはその「実物見学」版です。

## 停留所 0 ── まず「契約」を見る（言語の素振り）
Prism の関数の署名は、型・作用・失敗・時間を一か所に書いた**嘘のつけない契約書**です。検査器がそれを強制します。

| 検索 | ボタン | 何に注目するか |
|---|---|---|
| `pipe`    | Run | `5 \|> double \|> inc` ── パイプは `f(x)` の糖衣。型も作用も失敗も素通り |
| `effects` | Run | `!io` が `!console` を包含。**作用が署名に出る**（効果の粒度） |
| `collide` | Run | **多重ディスパッチ** ── 結果が両引数の型で決まる |

> ここで「署名を読めば、何に触れ・どう失敗しうるかが分かる」感覚を掴むのがゴール。詳しくは TUTORIAL。

## 停留所 1 ── 一枚の絵は「純粋な値」
`picture` は `List[Shape]`（絶対座標の図形のリスト）。I/O も状態もありません。`slider(...)` で対話に。

| 検索 | ボタン | 何に注目するか |
|---|---|---|
| `rose`                 | Draw | 1本の `sin` 曲線だけでこの複雑さ |
| `spirograph-studio`    | Draw | スライダが出る ── 値を動かすと絵が変わる（対話＝純粋なパラメータ） |
| `harmonograph-studio`  | Draw | 減衰する振り子の重ね合わせ |
| `phyllotaxis`          | Draw | 黄金角だけで葉序（ひまわりの種）が並ぶ |

## 停留所 4 ── 時間は `frame(t)`（アニメ）
アニメは**時間の純粋な関数**。遊び場が `t` を進めるだけで、状態も副作用もありません。

| 検索 | ボタン | 何に注目するか |
|---|---|---|
| `pendulum-anim`  | Animate | `frame(t)` が返す絵を時間で並べるだけ |
| `fourier-anim`   | Animate | 円を重ねて方形波を描く（フーリエ級数） |
| `wave3d-anim`    | Animate | 3D 波面の投影 |

## 停留所 5 ── ゲームエフェクトの正体
派手なVFXも、数パターンの組み合わせです。**残像＝過去の `t−δ` の位置を薄く重ねる**、火花＝放射＋重力＋減衰、乱数＝`sin`ハッシュ、発光＝半透明の塗り＋明るい芯。

| 検索 | ボタン | 何に注目するか |
|---|---|---|
| `magic-circle` | Animate | 回転リング＋ルーン＋発光 |
| `slash-trail`  | Animate | 斬撃の三日月＝モーションブラー（残像） |
| `sparkle-burst`| Animate | 命中エフェクト：放射＋重力＋瞬き |
| `combo`        | Animate | 7セグ数字（`lib/digits`）で作るコンボ表示 |
