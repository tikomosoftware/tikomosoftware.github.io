# 02 — 技術ハイライト（具体的に何を作ったか）

数字や抽象論ではなく、「実際に何が動いているか」を具体的に。コードの場所も添えます。

## 検査器（`check.py`）が強制していること

これが Prism の心臓です。署名と本体が食い違うと、行番号つきで拒否します。

| 検査 | 例（これを書くと怒られる） |
|---|---|
| **型の不一致** | 戻り値・引数・リスト要素・match の各腕が型として合わない |
| **副作用の宣言漏れ** | `!console` を使うのに署名が「純粋」 |
| **効果の粒度** | `!io` は `!console` を包含（一方向）。粗い宣言で細かい作用は可、逆はリーク扱い |
| **失敗の宣言漏れ／未処理** | 失敗しうる値を `try`/`match` せず使う、署名に `?Error` が無い |
| **網羅性** | `or` 型の場合分け漏れ、リストの空/非空、`ok`/`fail` の片方漏れ |
| **能力（trait）** | `given T: Cap` の未充足、`provides` のメソッド過不足、本体が署名違反 |
| **多重ディスパッチ** | `capability C for A, B` で、組合せにインスタンスが無い呼び出し |
| **グラデュアル型** | 署名を省いた所だけ緩み、*既知の型どうしの衝突*は弾く |
| **構築完全性** | 宣言済みレコードのフィールド欠落（`Pt{x:3}`） |
| **名前衝突** | `include` 併合で同名の二重定義 |
| **隠れた強制変換**（実行時） | `"ab" * 2` の黙った文字列反復などを拒否 |

「効果と失敗を*行多相*で追う」のが特に効いていて、**1つの `map` が、純粋・副作用つき・可謬な関数すべてに
対して正直なまま**使えます。これは本格的な型システムの機能です。

## 言語の機能セット

- 4つの声＝**事実 `:`/`and`/`or`・変換 `<-`/`=>`/`\|>`・作用 `!world`・失敗 `?Error`** ＋ 上級の **時間 `~>`**
- 代数的データ型（`or`）＋ レコード（`and`）＋ 網羅 match
- ジェネリクス（型・効果・失敗の穴）＋ **高階種 `F[_]`**（Functor など）
- 能力（capability/given/provides）＝ trait/typeclass、**多重ディスパッチ**（多引数の能力）
- グラデュアル型（`_` 境界）、パイプ `\|>`、`--reveal`（署名なし関数の契約を推論して表示）

## アーキテクチャ（小ささの理由）

```
prism.py  (1,157行)  字句解析 + 構文解析 + ツリーウォーク評価器（run の中身）
check.py  (759行)    静的検査器（型/効果/失敗/網羅/能力/高階種/時間 + --reveal）
                     合計 約1,900行・依存ゼロ・ビルド不要
```

- **ツリーウォーク評価器**：最も単純な実行方式。コンパイルもVMも無い。
- **検査器は評価器と完全に分離**（"missing organ" として後付けできる設計）。
- 数値・文字列・メモリ・例外は Python に委譲。**言語固有の本質だけ**を書いている。
- 新機能が既存の `unify`/`subst`（型単一化・置換）に載るので、行数が増えにくい。

## ブラウザ実行

- `playground.html` が Pyodide 上で**同じ `prism.py`/`check.py`** を実行（移植なし）。
- 起動時に `lib/*.prism` を仮想FSへ書き込み、`include` がそのまま解決。
- 300枚のギャラリーを Run / Draw / Animate で動かせる。

## 規模と検証

- 回帰テスト **824**（`prism test core` は言語コアのみ **99・約1〜2秒**、`gallery` は重い視覚物、`all` は全部）。
- ギャラリー **300枚**（物理230・フラクタル63ほか）。**全部が check 緑＋非空を機械検証**。
- 計算系デモ（素数・Rule110/30・ソート・ライフゲーム）は **Python 参照と一致を検証**。

## 標準ライブラリ（`lib/`、12本・include 可能）

`list`/`math`/`picture`（図形代数）/`physics2d`/`draw2d`/`turtle`/`lsystem`/`escape`（複素数・脱出時間）/
`plot`/`chart`/`ca`（セルオートマトン）/`digits`（7セグ数字）。
**言語は閉じていて、育てるのはライブラリ**、という方針がそのまま形になっています。
