Docs

主に Claude Code を対象にした、AI活用・AI開発支援の導入ガイドです。まだ書きかけのところも多いですが、少しずつ育てています。

この資料を作った経緯

個人で開発しているぶんには、AIを好き勝手に使えてとても快適です。 でも複数メンバーになった途端、コストが一気に上がるな、と感じていました。 誰がどう使っているかわからない、レビューの基準がバラバラ、情報の扱いも人によって違う。 ある程度ルールを決めないと、AIを使っているのに逆に手間が増える、という状況になりかねない。 そう思って、チームで使うための考え方や手順をまとめてみました。
